Introduction

Opspec (operational specification) is a declarative language designed for the sole purpose of describing operations (ops).

#Types

Values in opspec are typed.

Reference:

  • array
  • boolean
  • dir
  • file
  • number

  • socket

#Scoping

Variables in opspec are scoped to each operation. Parameters allow passing values into or out of this scope.
