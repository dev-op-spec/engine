String Parameter [object]

An object defining a parameter which accepts a string typed value.

#Properties:

  • must have:
    • description
  • may have:
    • constraints
    • default
    • isSecret

#constraints

A JSON Schema v4 [object] defining constraints on the parameters value.

#default

A string to use as the value of the parameter when no argument is provided.

#description

A markdown [string] defining a human friendly description of the parameter.

#isSecret

A boolean indicating if the value of the parameter is secret. This will cause it to be hidden in UI's for example.
