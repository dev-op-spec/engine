## Examples

Deploy opctl in kubernetes

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: opctl-in-kubernetes
spec:
  replicas: 1
  template:
    spec:
      containers:
        - name: opctl
          image: opctl/opctl:0.1.39-dind
          ports:
            # expose to other containers
            - name: http
              containerPort: 42224
              protocol: TCP
          securityContext:
            privileged: true
```
